1. What technologies do you use for web development?

I use modern front-end frameworks like React or Vue.js, combined with back-end technologies such as Node.js, Express, and databases like MongoDB or PostgreSQL.

2. Do you build cross-platform mobile apps?

Yes, I develop both native and cross-platform apps using tools like Flutter, React Native, and Kotlin/Swift when needed.

3. What types of embedded systems have you worked on?

I’ve built projects using microcontrollers (like Arduino, STM32, ESP32) for applications in IoT, automation, and real-time sensor-based systems.
